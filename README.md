# Media Data

![Mostly decorative banner that displays a screenshot of the Media Data block in action and Nova, the brand's mascot, in a city holding a camera.](/.wporg/banner-1544x500.jpg)

Display data about any media file uploaded to WordPress via the **Media Data** block.

## To Install/Test

This plugin is currently under development. To test, you must have Composer on your machine. Once you clone the repo, be sure to run the following commands before activating the plugin:

```bash
composer install
composer build
```
